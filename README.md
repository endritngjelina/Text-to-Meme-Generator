# Text-to-Meme-Generator

Create memes from a simple text prompt — fast and visually clean.

This repository contains a set of **HTML prototypes** that explore different design iterations for a *Text‑to‑Meme Generator*. These versions test UI/UX structures for allowing a user to input text, place captions, and preview a meme layout. Each `.html` file represents a standalone iteration.

> **Purpose:** These prototypes are meant for layout exploration and instruction. They do not yet generate real images but provide the foundation for a future working version.

---

## ✨ Features (Prototype Scope)

- **Multiple UI iterations:** Versions `1.html` through `7.html`, each with unique design/layout tweaks.
- **Instant previewing:** Open any file directly in your browser — no dependencies.
- **Simple, readable structure:** Clear HTML intended for teaching, reviewing, and improvement.
- **MIT License:** Fully open for modification and reuse.
